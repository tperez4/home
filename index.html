Video Guestbook - Java Application

Role: Lead Software Developer

Key Technologies: Java, Eclipse, Adobe Photoshop CS6

The Video Guestbook is a Java application. It allows users to record a video message using a webcam connected to the terminal. Users can review their video message, re-record, or delete their entry before submission. Users can also view entries which were submitted by previous users.

Responsibilities & Achievements

  • Structured the application according to Object Oriented programming design principles
  • Implemented timer functions to ensure the program can run in an unsupervised environment

  • Demonstrated program functionality and usage to various clientele and companies

RUSocial - Android Application

Role: Software Developer, UI Developer

Key Technologies: Java, Android SDK, Google Maps API, Agile

RUSocial was created exclusively for Ryerson students. It is an Android-based social media app that allows users to interact with both their friends on campus and with the Ryerson community. By integrating the dynamic features of Google maps, users are able to interact with their Ryerson peers like never before.

Responsibilities & Achievements

  • Learned and programmed with the Android SDK and the Google Maps API

  • Created application graphics and logos using Adobe Photoshop CS6
